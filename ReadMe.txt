Git is a free version control system:

Git initialize repository command:
  git init       
    ## 初始化一个 Repository

  git add <file> 
    ## 添加一个文件至 Repository
  
  git commit -m <message>
    ## 提交当前改动至 Repository

  git status
    ## 获取当前 Tree 的状态。
    ##   对于改动了的文件，最终还是要
    ##      git add <file>
    ##      git commit -m <message>
